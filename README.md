# GoToCoin
3D video game about a player who has to run to get coins and win the game.

## Characteristics
- Type: 3D
- Point of view: First person
- Gender: First Person Runner
- Platform: 3D
- Graphics engine: Unity 3D
- Main language: C#
- Level : 1
